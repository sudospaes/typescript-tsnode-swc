# Node.js TypeScript (TS-Node + SWC) Template
A Node.js project template using TypeScript with TS-Node + SWC compiler and configured to CommonJS.

## Preparation for developing :

1 - Install packages.
```bash
npm i
```
2 - Run to develop
```bash
npm run dev
```

## Run for production :
1 - Make a build from project
```bash
npm run build
```
2 - Run project
```bash
npm start
```
